# Page d'accueil de Louise

Ma page perso avec les liens vers tous mes projets.

**Voir en ligne :** https://loulouge.github.io/landing-page/

## Mes projets

- **Vocabulaire** — Apprends du vocabulaire anglais-français avec des quiz et des jeux d'association
- **Calcul mental** — Entraîne-toi au calcul mental
- **Empereurs romains** — Découvre et apprends les empereurs romains
- **Chicken Geo** — Teste tes connaissances en géographie

## Lancer le projet

Ouvre `index.html` dans ton navigateur, ou lance un serveur local :

```bash
python3 -m http.server 8080
```

Puis ouvre `http://localhost:8080`.

## Technologies

HTML, CSS, JavaScript (vanilla) — aucune dépendance externe.
